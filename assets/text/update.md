## v0.1.0 (26.01.16)
### 1. 검색창 기능 제거
- 검색 창 기능을 제거하고, 작품 추가 버튼으로 변경
- 작품 추가 버튼은 검정색의 원형 배경에 + 아이콘으로 시각적 강조
- 작품 추가 버튼을 눌렀을 때, 작품 추가 기능이 뜨도록 변경
- 기존 [홈] 창에서 작품 태그 실행 시 [검색]창으로 이동하는 로직 제거.

### 2. [커뮤니티] 탭 기능 변경
- [커뮤니티] 탭의 이름을 [아카이브]로 변경, 적절한 아이콘으로 변경.
- [아카이브] 페이지 상단에 [피드], [달력], [친구] 탭 생성
- [아카이브] 페이지 클릭 시, [피드] 탭이 기본으로 선택되어 표출되도록 구현
    - [피드] 탭 클릭 시, 기존 [커뮤니티] 기능과 동일하게 구현
        - 팔로우한 친구의 게시물을 확인할 수 있도록 구현 
        - 본인이 업로드한 게시물은 제외, 본인이 저장한 작품은 [작품] 페이지에서 확인
        - [피드] 창에서 활동한 기록(좋아요, 댓글, 북마크)은 [마이]-[활동]창에서 확인할 수있도록 설계
        - 북마크 기능 추가
    - [달력] 탭 클릭 시, 달력 위젯 표출
        - 작품을 등록한 날짜를 확인할 수 있도록 구현
        - 작품을 등록한 날짜에 등록한 작품이 표출되도록 구현
        - 특정 날짜에 2개 이상의 작품을 등록한 경우, +n 표시를 통해 추가 등록 작품이 있다는 것을 시각적으로 나타내도록 구현 (단, 이때의 n은 임의의 정수를 말함.)
        - 작품이 등록되지 않은 경우, 회색 배경의 날짜 표시를 통해 작품이 등록되지 않았음을 시각적으로 나타내도록 구현.
    - [친구] 탭 클릭 시, 친구 관리 창 표출
        - 친구 검색 기능 구현 및 친구 요청 버튼 구현
        - 현재 팔로우 중인 친구 목록 표출
        - 팔로우 취소 버튼 구현

### 3. 검색 기능 개편
- [홈] 창의 검색 바에 검색어 입력 시, 검색어에 해당하는 작품이 [홈] 창에 표출되도록 구현.
- [태그] 및 [별점] 버튼 클릭 시, [검색] 기능과 동일하게 해당하는 작품이 [홈] 창에 표출되도록 구현.

### 4. [마이] 페이지 개편
- 프로필 사진 기능 추가
- 프로필 구성 (중앙 정렬)
    - 프로필 사진
    - 닉네임
    - 한줄 소개 (수정이 가능하도록 기능 구현)
    - 팔로워 / 팔로잉 수 표시
- 나의 활동 페이지 구현
    - 상단에 탭으로 구분
        - 북마크
        - 좋아요
        - 댓글
    - 북마크 : 
        - 북마크한 작품 목록(작품 이미지, 작품 제목, 작가, 평점, 좋아요 수, 댓글 수, 북마크한 날짜)
        - 본인이 등록한 작품 중 북마크한 작품도 포함
    - 좋아요 : 
        - 좋아요한 작품 목록(작품 이미지, 작품 제목, 작가, 평점, 좋아요 수, 댓글 수, 좋아요한 날짜)
        - 본인이 등록한 작품 중 좋아요한 작품도 포함
    - 댓글 : 
        - 댓글을 남긴 작품 목록(작품 이미지, 작품 제목, 작가, 평점, 좋아요 수, 댓글 수, 댓글한 날짜)
        - 본인이 등록한 작품 중 댓글을 남긴 작품도 포함
- 계정 설정 페이지 구현(비밀번호 변경, 탈퇴)
